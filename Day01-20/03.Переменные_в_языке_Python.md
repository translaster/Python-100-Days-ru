## Переменные в языке Python

Для новичков, которые хотят научиться программировать, есть два вопроса, один из которых - «что такое (компьютерная) программа», а другой - «что я могу сделать, написав (компьютерную) программу». Давайте начнем с моего понимания этих двух вопросов: **Программа - это упорядоченная коллекция данных и инструкций**, а **написать программу - значит управлять компьютером с помощью данных и инструкций, чтобы он делал то, что мы хотим**. Причина, по которой так много людей сегодня предпочитают писать программы на Python, заключается в том, что Python простой и достаточно мощный. По сравнению с такими языками программирования, как C, C++ и Java, Python гораздо более дружелюбен к новичкам и неспециалистам, и многие проблемы могут быть решены на Python просто и элегантно. Давайте начнем с основ языка и познакомимся с Python поближе.

### Немного здравого смысла

Прежде чем приступить к систематическому изучению программирования на Python, давайте сначала познакомимся с основами компьютерной грамотности. Аппаратная система компьютера обычно состоит из пяти основных компонентов, включая: **процессор**, **контроллер**, **память**, **устройство ввода** и **устройство вывода**. Процессор и контроллер вместе представляют собой то, что мы часто называем **центральным процессором** (CPU), функции которого заключаются в выполнении различных арифметических и управляющих инструкций. Как мы уже говорили, программа - это набор инструкций, а написание программы - это организация серии инструкций определенным образом, а затем использование этих инструкций для управления компьютером, чтобы сделать то, что мы хотим, чтобы он сделал. Память можно разделить на **внутреннюю** и **внешнюю**. Первая - это то, что мы часто называем памятью, то есть пространство для хранения данных, к которому может напрямую обращаться центральный процессор, и которое соответствует данным и инструкциям, которые необходимо загрузить в память во время выполнения программы. Устройства ввода и устройства вывода часто называют устройствами ввода-вывода. Клавиатуры, мыши, микрофоны и камеры являются типичными устройствами ввода, а мониторы, принтеры, колонки и т.д. - типичными устройствами вывода. В настоящее время большинство компьютеров, которыми мы пользуемся, по сути, являются компьютерами, построенными по «архитектуре фон Неймана», которая имеет два ключевых момента: первый - это **отделение памяти от центрального процессора**; второй - **кодирование данных в двоичном виде**.

Двоичный метод счета - это «двоичный» метод счета, который по сути является тем же самым, что и «десятичный» метод счета, используемый людьми. Люди используют десятичную систему счисления, потому что у них десять пальцев, и когда пальцы заканчиваются, они могут использовать только округление для представления больших значений. Конечно, есть исключения из правил, майя, возможно, много лет ходили босиком, потому что пальцы на ногах тоже используются, поэтому они использовали двоичный метод счета. Основываясь на этом способе счета, календарь майя отличался от того, которым мы обычно пользуемся. Согласно календарю майя, 2012 год был последним годом последней так называемой «солнечной эры», а 2013-й - началом новой «солнечной эры». Впоследствии это было искажено как пророчество майя о конце света в 2012 году. Сегодня существует множество предположений, что медленное развитие цивилизации майя было как-то связано с использованием двоичной системы счисления. Для компьютеров двоичную систему проще всего реализовать на физическом устройстве, поскольку она может использоваться для представления 1 с высоким напряжением и 0 с низким напряжением. Не все программисты должны знать двоичную систему, преобразование десятичной системы в двоичную, восьмеричную и шестнадцатеричную, но чаще всего мы можем писать программы, даже не имея этих знаний. Однако мы должны знать, что компьютер использует двоичный счет, независимо от того, какие данные поступают в память компьютера именно в двоичной форме.

> **Примечание**: О методе двоичного счета и о том, как он преобразуется в другие системы, вы можете перелистать книгу под названием «Введение в компьютер» или «Компьютерная культура», в ней вы найдете соответствующие знания, здесь это не будет проводиться для повторения читателю неясные для него собственные исследования.

### Переменные и типы

Чтобы сохранить данные в памяти компьютера, необходимо сначала поговорить о понятии переменной. В языках программирования **переменная - это носитель данных**, просто участок памяти, используемый для хранения данных, значение **переменной может быть прочитано и изменено**, что является основой всех операций и управления. Существует множество типов данных, с которыми могут работать компьютеры, наиболее распространенными из них являются числовые значения, а в дополнение к числовым значениям существует широкий спектр типов данных, таких как текст, изображения, аудио, видео и так далее. Хотя в компьютерах данные существуют в двоичной форме, мы можем использовать различные типы переменных для представления различий в типах данных, а язык Python поставляется с широким спектром предопределенных типов данных, а также возможностью создавать новые типы данных, о которых мы поговорим позже. Для начала давайте рассмотрим некоторые из наиболее распространенных типов данных, используемых в Python.

1. Целочисленный тип (`int`): Python может работать с целыми числами любого размера и поддерживает двоичное (например, `0b100`, которое преобразуется в 4), восьмеричное (например, `0o100`, которое преобразуется в 64), десятичное (`100`) и шестнадцатеричное (`0x100`, которое преобразуется в 256) представления. Запустите приведенный ниже код и посмотрите, что будет выведено.

    ```python
    print(0b100)  # Двоичные целые числа
    print(0o100)  # Целые восьмеричные числа
    print(100)    # Десятичные целые числа
    print(0x100)  # Шестнадцатеричные целые числа
    ```

2. Вещественный тип (`float`): числа с плавающей точкой, также известные как десятичные, называются числами с плавающей точкой, потому что при выражении в научной нотации число с плавающей точкой имеет переменный десятичный знак, и числа с плавающей точкой поддерживают научную нотацию в дополнение к математической записи (например, `123.456`) (например, `1.23456e2`, что означает $\small{1.23456 \times 10^{2}}$). Запустите следующий код и посмотрите, что будет выведено.

    ```python
    print(123.456)    # 数学写法
    print(1.23456e2)  # 科学计数法
    ```

3. Строковый тип (`str`): строка - это любой текст, заключенный в одинарные или двойные кавычки, например, `'hello'` и `"hello"`.

4. Логический тип (`bool`): имеет только два значения, `True` и `False`, либо `True`, либо `False`, и может быть использовано для представления `Yes` и `No` в реальном мире, и `True` и `False` для предложений. "True" и "False" для пропозиций, "Good" и "Bad" для ситуаций, "High" и "Low" для уровней, а также "High" и "Low" для уровней и так далее. Если значение переменной имеет только два состояния, мы можем использовать логический тип.

### Именование переменных

Каждой переменной нужно дать имя, как и каждому из нас. В Python именование переменных должно соответствовать следующим правилам и соглашениям.

- Раздел «Правила»:
  - Правило 1: Имя переменной состоит из **букв**, **чисел** и **подчеркиваний** и не может начинаться с цифры. Следует отметить, что буквы, упомянутые здесь, относятся к символам Unicode. Unicode известен как универсальный код, который охватывает большинство мировых систем письма, это означает, что китайские, японские, греческие буквы и т.д. могут быть использованы в качестве символов в именах переменных, но некоторые специальные символы (например, `!`, `@`, `#` и т. д.) недопускаются. Мы настоятельно рекомендуем интерпретировать упомянутые здесь буквы как **Используйте только английские буквы**, когда это возможно.
  - Правило 2: Python - это **чувствительный к регистру** язык программирования, что просто означает, что заглавная `A` и строчная `a` - это две разные переменные; это не совсем правило, но скорее то, что следует иметь в виду.
  - Правило 3: Имена переменных **не переименовываются с помощью ключевых слов Python**, и **по возможности избегайте зарезервированных слов Python**. Ключевые слова - это слова, которые имеют специальные значения в программах на Python (например, `is`, `if`, `else`, `for`, `while`, `True`, `False` и т.д.), а зарезервированные слова - это имена встроенных функций Python, встроенных модулей и т.д. (например, `int`, `print`, `input`, `str`, `math` и т.д.).
- Раздел «Условные обозначения»:
  - Соглашение 1: В именах переменных обычно используются **строчные английские буквы**, **многозначные слова объединяются подчеркиванием**.
  - Конвенция 2: Защищенные переменные начинаются с одного подчеркивания.
  - Конвенция 3: Частные переменные начинаются с двух знаков подчеркивания.

На данный момент соглашения 2 и 3 следует игнорировать, поскольку они станут понятны позже. Конечно, как профессиональному программисту, также очень важно помнить о названии переменных.  Это демонстрирует профессиональный нрав программиста, и собеседования на многие должности разработчиков также очень ценят это.

### Использование переменных

Следующие примеры иллюстрируют типы переменных и их использование.

```python
"""
Использование переменные для хранения данных и выполнения операций сложения, вычитания, умножения и деления

Version: 1.0
Author: Ло Хао
"""
a = 45        # Определить переменную a и присвоить ей значение 45
b = 12        # Определить переменную b и присвоить ей значение 12
print(a, b)   # 45 12
print(a + b)  # 57
print(a - b)  # 33
print(a * b)  # 540
print(a / b)  # 3.75
```

В Python вы можете использовать функцию `type` для проверки типа переменной. Понятие функции в программировании очень похоже на понятие функции в математике, с которым, я уверен, вы знакомы, и состоит из имени функции, независимой переменной и зависимой переменной. Если вы не понимаете концепцию функции, это неважно, мы рассмотрим в последующих материалах, посвященных определению и использованию функции.

```python
"""
Проверка типа переменной с помощью функции type

Version: 1.0
Author: Ло Хао
"""
a = 100
b = 123.45
c = 'hello, world'
d = True
print(type(a))  # <class 'int'>
print(type(b))  # <class 'float'>
print(type(c))  # <class 'str'>
print(type(d))  # <class 'bool'>
```

Вы можете изменить тип переменной с помощью встроенных функций Python. Вот некоторые распространенные функции, связанные с типами переменных.

- `int()`: преобразует числовое значение или строку в целое число, по возможности с указанием системы счисления.
- `float()`: преобразует строку (где это возможно) в число с плавающей точкой.
- `str()`: преобразует указанный объект в строковую форму, кодировка может быть указана.
- `chr()`: преобразует целое число (в символьной кодировке) в соответствующую (односимвольную) строку.
- `ord()`: преобразует (односимвольную) строку в соответствующее целое число (кодировка символов).

Следующий пример демонстрирует преобразование типов в Python.

```python
"""
Операции преобразования типов переменных

Version: 1.0
Author: Ло Хао
"""
a = 100
b = 123.45
c = '123'
d = '100'
e = '123.45'
f = 'hello, world'
g = True

print(float(a))         # тип int 100 в float, вывод 100.0
print(int(b))           # тип float 123.45 в int, вывод 123
print(int(c))           # str тип '123' в int, вывод 123
print(int(c, base=16))  # строка типа '123', преобразованная в int в шестнадцатеричном виде, вывод 291
print(int(d, base=2))   # str тип '100' преобразуется в int в двоичном виде, вывод 4
print(float(e))         # строковый тип '123.45', преобразованный в float, вывод 123.45
print(bool(f))          # строковый тип 'hello, world' преобразуется в bool, вывод True
print(int(g))           # bool типа True в int, вывод 1
print(chr(a))           # int типа 100 в str, вывод 'd'
print(ord('d'))         # тип str 'd' в int, вывод 100
```

> **Описание**: Тип `str` к типу `int` может быть указан параметром `base`, который может быть преобразован путем обработки строки как целого числа соответствующей системы. При преобразовании типа `str` в тип `bool`, пока строка имеет содержимое, а не `''` или `""`, соответствующее булево значение будет `True`. Когда типы `bool` преобразуются в типы `int`, `True` становится `1`, а `False` - `0`. Как в наборе символов ASCII, так и в наборе символов Unicode символ `'d'` соответствует `100`.
