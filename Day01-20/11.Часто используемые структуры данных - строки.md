## –ß–∞—Å—Ç–æ –∏—Å–ø–æ–ª—å–∑—É–µ–º—ã–µ —Å—Ç—Ä—É–∫—Ç—É—Ä—ã –¥–∞–Ω–Ω—ã—Ö - —Å—Ç—Ä–æ–∫–∏

–í—Ç–æ—Ä–∞—è –º–∏—Ä–æ–≤–∞—è –≤–æ–π–Ω–∞ –ø—Ä–∏–≤–µ–ª–∞ –∫ —Ä–æ–∂–¥–µ–Ω–∏—é —Å–æ–≤—Ä–µ–º–µ–Ω–Ω–æ–≥–æ —ç–ª–µ–∫—Ç—Ä–æ–Ω–Ω–æ–≥–æ –∫–æ–º–ø—å—é—Ç–µ—Ä–∞. –ü–µ—Ä–≤—ã–π –≤ –º–∏—Ä–µ —ç–ª–µ–∫—Ç—Ä–æ–Ω–Ω—ã–π –∫–æ–º–ø—å—é—Ç–µ—Ä –æ–±—â–µ–≥–æ –Ω–∞–∑–Ω–∞—á–µ–Ω–∏—è –ø–æ–¥ –Ω–∞–∑–≤–∞–Ω–∏–µ–º ENIAC (Electronic Numerical Integration Calculator) –ø–æ—è–≤–∏–ª—Å—è –≤ –°–®–ê –≤ –ü–µ–Ω—Å–∏–ª—å–≤–∞–Ω—Å–∫–æ–º —É–Ω–∏–≤–µ—Ä—Å–∏—Ç–µ—Ç–µ, –∑–∞–Ω–∏–º–∞–ª –ø–ª–æ—â–∞–¥—å 167 –∫–≤–∞–¥—Ä–∞—Ç–Ω—ã—Ö –º–µ—Ç—Ä–æ–≤, –≤–µ—Å–∏–ª –æ–∫–æ–ª–æ 27 —Ç–æ–Ω–Ω –∏ —Å–ø–æ—Å–æ–±–µ–Ω –±—ã–ª –≤—ã–ø–æ–ª–Ω—è—Ç—å –æ–∫–æ–ª–æ 5 000 –æ–ø–µ—Ä–∞—Ü–∏–π —Å –ø–ª–∞–≤–∞—é—â–µ–π —Ç–æ—á–∫–æ–π –≤ —Å–µ–∫—É–Ω–¥—É, –∫–∞–∫ –ø–æ–∫–∞–∑–∞–Ω–æ –Ω–∞ —Ä–∏—Å—É–Ω–∫–µ –Ω–∏–∂–µ. ENIAC –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–ª—Å—è –¥–ª—è —Ä–∞—Å—á–µ—Ç–∞ —Ç—Ä–∞–µ–∫—Ç–æ—Ä–∏–∏ –ø–æ–ª–µ—Ç–∞ —Ä–∞–∫–µ—Ç, –∏ —è–≤–ª—è–ª—Å—è –æ–¥–Ω–æ–π –∏–∑ –≤–∞–∂–Ω–µ–π—à–∏—Ö —Ñ—É–Ω–∫—Ü–∏–π —Å–æ–≤—Ä–µ–º–µ–Ω–Ω–æ–≥–æ —ç–ª–µ–∫—Ç—Ä–æ–Ω–Ω–æ–≥–æ –∫–æ–º–ø—å—é—Ç–µ—Ä–∞. –ß–∏—Å–ª–µ–Ω–Ω—ã–µ –≤—ã—á–∏—Å–ª–µ–Ω–∏—è - –æ–¥–Ω–∞ –∏–∑ –≤–∞–∂–Ω–µ–π—à–∏—Ö —Ñ—É–Ω–∫—Ü–∏–π —Å–æ–≤—Ä–µ–º–µ–Ω–Ω—ã—Ö –∫–æ–º–ø—å—é—Ç–µ—Ä–æ–≤.

–°–æ –≤—Ä–µ–º–µ–Ω–µ–º, —Ö–æ—Ç—è —á–∏—Å–ª–æ–≤—ã–µ –æ–ø–µ—Ä–∞—Ü–∏–∏ –æ—Å—Ç–∞—é—Ç—Å—è –Ω–∞–∏–±–æ–ª–µ–µ –≤–∞–∂–Ω–æ–π —á–∞—Å—Ç—å—é –ø–æ–≤—Å–µ–¥–Ω–µ–≤–Ω–æ–π —Ä–∞–±–æ—Ç—ã –∫–æ–º–ø—å—é—Ç–µ—Ä–∞, —Å–æ–≤—Ä–µ–º–µ–Ω–Ω—ã–µ –∫–æ–º–ø—å—é—Ç–µ—Ä—ã —Ç–∞–∫–∂–µ –∏–º–µ—é—Ç –¥–µ–ª–æ —Å –±–æ–ª—å—à–∏–º –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ–º –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏–∏ –≤ –≤–∏–¥–µ —Ç–µ–∫—Å—Ç–∞. –ï—Å–ª–∏ –º—ã —Ö–æ—Ç–∏–º –º–∞–Ω–∏–ø—É–ª–∏—Ä–æ–≤–∞—Ç—å —ç—Ç–æ–π —Ç–µ–∫—Å—Ç–æ–≤–æ–π –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏–µ–π —Å –ø–æ–º–æ—â—å—é –ø—Ä–æ–≥—Ä–∞–º–º –Ω–∞ Python, –º—ã –¥–æ–ª–∂–Ω—ã —Å–Ω–∞—á–∞–ª–∞ –ø–æ–Ω—è—Ç—å —Ç–∏–ø –¥–∞–Ω–Ω—ã—Ö string –∏ —Å–≤—è–∑–∞–Ω–Ω—ã–µ —Å –Ω–∏–º –æ–ø–µ—Ä–∞—Ü–∏–∏ –∏ –º–µ—Ç–æ–¥—ã.

### –û–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–µ —Å—Ç—Ä–æ–∫

–¢–∞–∫ –Ω–∞–∑—ã–≤–∞–µ–º–∞—è **—Å—Ç—Ä–æ–∫–∞** - —ç—Ç–æ **–∫–æ–Ω–µ—á–Ω–∞—è –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç—å** –∏–∑ –Ω—É–ª—è –∏–ª–∏ –±–æ–ª–µ–µ —Å–∏–º–≤–æ–ª–æ–≤, –æ–±—ã—á–Ω–æ –æ–±–æ–∑–Ω–∞—á–∞–µ–º–∞—è –∫–∞–∫:

$$
s = a_1a_2 \cdots a_n \,\,\,\,\, (0 \le n \le \infty)
$$

–í –ø—Ä–æ–≥—Ä–∞–º–º–∞—Ö –Ω–∞ Python –º—ã –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è–µ–º —Å—Ç—Ä–æ–∫—É, –∑–∞–∫–ª—é—á–∞—è –æ–¥–∏–Ω –∏–ª–∏ –Ω–µ—Å–∫–æ–ª—å–∫–æ —Å–∏–º–≤–æ–ª–æ–≤ –≤ –æ–¥–∏–Ω–∞—Ä–Ω—ã–µ –∏–ª–∏ –¥–≤–æ–π–Ω—ã–µ –∫–∞–≤—ã—á–∫–∏. –°–∏–º–≤–æ–ª–∞–º–∏ –≤ —Å—Ç—Ä–æ–∫–µ –º–æ–≥—É—Ç –±—ã—Ç—å —Å–ø–µ—Ü–∏–∞–ª—å–Ω—ã–µ —Å–∏–º–≤–æ–ª—ã, –∞–Ω–≥–ª–∏–π—Å–∫–∏–µ –±—É–∫–≤—ã, –∫–∏—Ç–∞–π—Å–∫–∏–µ –∏–µ—Ä–æ–≥–ª–∏—Ñ—ã, —è–ø–æ–Ω—Å–∫–∞—è —Ö–∏—Ä–∞–≥–∞–Ω–∞ –∏–ª–∏ –∫–∞—Ç–∞–∫–∞–Ω–∞, –≥—Ä–µ—á–µ—Å–∫–∏–µ –±—É–∫–≤—ã, —Å–∏–º–≤–æ–ª—ã Emoji (–Ω–∞–ø—Ä–∏–º–µ—Ä, üí©, üê∑, üÄÑÔ∏è) –∏ —Ç–∞–∫ –¥–∞–ª–µ–µ.

```python
s1 = 'hello, world!'
s2 = "–ü—Ä–∏–≤–µ—Ç, –º–∏—ÄÔºÅ‚ù§Ô∏è"
s3 = '''hello,
wonderful
world!'''
print(s1)
print(s2)
print(s3)
```

#### –≠—Å–∫–µ–π–ø-—Å–∏–º–≤–æ–ª—ã

–ú—ã –º–æ–∂–µ–º –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å `\` (–æ–±—Ä–∞—Ç–Ω—ã–π —Å–ª–µ—à) –≤ —Å—Ç—Ä–æ–∫–µ –¥–ª—è –æ–±–æ–∑–Ω–∞—á–µ–Ω–∏—è —ç–∫—Ä–∞–Ω–∏—Ä–æ–≤–∞–Ω–∏—è, —ç—Ç–æ –æ–∑–Ω–∞—á–∞–µ—Ç, —á—Ç–æ —Å–∏–º–≤–æ–ª –ø–æ—Å–ª–µ `\` –±–æ–ª—å—à–µ –Ω–µ –∏–º–µ–µ—Ç —Å–≤–æ–µ–≥–æ –ø–µ—Ä–≤–æ–Ω–∞—á–∞–ª—å–Ω–æ–≥–æ –∑–Ω–∞—á–µ–Ω–∏—è, –Ω–∞–ø—Ä–∏–º–µ—Ä, `\n` –æ–±–æ–∑–Ω–∞—á–∞–µ—Ç –Ω–µ —Å–∏–º–≤–æ–ª `\` –∏ —Å–∏–º–≤–æ–ª `n`, –∞ –Ω–æ–≤—É—é —Å—Ç—Ä–æ–∫—É; `\t` —Ç–∞–∫–∂–µ –æ–±–æ–∑–Ω–∞—á–∞–µ—Ç –Ω–µ —Å–∏–º–≤–æ–ª `\` –∏ —Å–∏–º–≤–æ–ª `t`, –∞ —Ç–∞–±—É–ª—è—Ü–∏—é. –¢–∞–∫–∏–º –æ–±—Ä–∞–∑–æ–º, –µ—Å–ª–∏ —Å—Ç—Ä–æ–∫–∞ —Å–æ–¥–µ—Ä–∂–∏—Ç `'`, `"`, `\` –∏ —ç—Ç–∏ —Å–ø–µ—Ü–∏–∞–ª—å–Ω—ã–µ —Å–∏–º–≤–æ–ª—ã, –æ–Ω–∞ –¥–æ–ª–∂–Ω–∞ –±—ã—Ç—å —ç–∫—Ä–∞–Ω–∏—Ä–æ–≤–∞–Ω–∞ `\`. –ù–∞–ø—Ä–∏–º–µ—Ä, —á—Ç–æ–±—ã –≤—ã–≤–µ—Å—Ç–∏ —Å—Ç—Ä–æ–∫—É —Å –æ–¥–∏–Ω–∞—Ä–Ω—ã–º–∏ –∫–∞–≤—ã—á–∫–∞–º–∏ –∏–ª–∏ –æ–±—Ä–∞—Ç–Ω—ã–º–∏ —Å–ª—ç—à–∞–º–∏, –Ω—É–∂–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –º–µ—Ç–æ–¥, –ø–æ–∫–∞–∑–∞–Ω–Ω—ã–π –Ω–∏–∂–µ.

```python
s1 = '\'hello, world!\''
s2 = '\\hello, world!\\'
print(s1)
print(s2)
```

#### –°—ã—Ä—ã–µ —Å—Ç—Ä–æ–∫–∏

–í Python –µ—Å—Ç—å —Å—Ç—Ä–æ–∫–∏, –Ω–∞—á–∏–Ω–∞—é—â–∏–µ—Å—è —Å `r` –∏–ª–∏ `R`, –∫–æ—Ç–æ—Ä—ã–µ –Ω–∞–∑—ã–≤–∞—é—Ç—Å—è —Å—ã—Ä—ã–º–∏ —Å—Ç—Ä–æ–∫–∞–º–∏, —Ç–æ –µ—Å—Ç—å –∫–∞–∂–¥—ã–π —Å–∏–º–≤–æ–ª –≤ —Å—Ç—Ä–æ–∫–µ –æ–∑–Ω–∞—á–∞–µ—Ç —Ç–æ, —á—Ç–æ –æ–Ω –¥–æ–ª–∂–µ–Ω –æ–∑–Ω–∞—á–∞—Ç—å, –±–µ–∑ —Ç–∞–∫ –Ω–∞–∑—ã–≤–∞–µ–º—ã—Ö escape-—Å–∏–º–≤–æ–ª–æ–≤. –ù–∞–ø—Ä–∏–º–µ—Ä, –≤ —Å—Ç—Ä–æ–∫–µ `'hello\n'` `\n` –æ–±–æ–∑–Ω–∞—á–∞–µ—Ç –ø–µ—Ä–µ–Ω–æ—Å —Å—Ç—Ä–æ–∫–∏, –∞ –≤ —Å—Ç—Ä–æ–∫–µ `r'hello\n'` `\n` —É–∂–µ –Ω–µ –æ–±–æ–∑–Ω–∞—á–∞–µ—Ç –ø–µ—Ä–µ–Ω–æ—Å —Å—Ç—Ä–æ–∫–∏, –∞ –æ–±–æ–∑–Ω–∞—á–∞–µ—Ç —Å–∏–º–≤–æ–ª \\\ –∏ —Å–∏–º–≤–æ–ª \n`. –í—ã –º–æ–∂–µ—Ç–µ –∑–∞–ø—É—Å—Ç–∏—Ç—å —Å–ª–µ–¥—É—é—â–∏–π –∫–æ–¥, —á—Ç–æ–±—ã –ø–æ—Å–º–æ—Ç—Ä–µ—Ç—å, —á—Ç–æ –±—É–¥–µ—Ç –≤—ã–≤–µ–¥–µ–Ω–æ.

```python
s1 = '\it \is \time \to \read \now'
s2 = r'\it \is \time \to \read \now'
print(s1)
print(s2)
```

> **–ü—Ä–∏–º–µ—á–∞–Ω–∏–µ**: –í –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–π `s1` –≤—ã—à–µ, `\t`, `\r` –∏ `\n` —è–≤–ª—è—é—Ç—Å—è —É–ø—Ä–∞–≤–ª—è—é—â–∏–º–∏ —Å–∏–º–≤–æ–ª–∞–º–∏. `\t` - —ç—Ç–æ —Ç–∞–±—É–ª—è—Ü–∏—è, `\n` - –Ω–æ–≤–∞—è —Å—Ç—Ä–æ–∫–∞, –∞ `\r` - –≤–æ–∑–≤—Ä–∞—Ç –∫–∞—Ä–µ—Ç–∫–∏, –∫–æ—Ç–æ—Ä—ã–π —ç–∫–≤–∏–≤–∞–ª–µ–Ω—Ç–µ–Ω –≤–æ–∑–≤—Ä–∞—Ç—É –≤—ã–≤–æ–¥–∞ –≤ –Ω–∞—á–∞–ª–æ —Å—Ç—Ä–æ–∫–∏. –°—Ä–∞–≤–Ω–∏—Ç–µ –≤—ã–≤–æ–¥ –¥–≤—É—Ö —Ñ—É–Ω–∫—Ü–∏–π `print`, —á—Ç–æ–±—ã —É–≤–∏–¥–µ—Ç—å —Ä–∞–∑–Ω–∏—Ü—É!

#### –°–ø–µ—Ü–∏–∞–ª—å–Ω–æ–µ –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–∏–µ —Å–∏–º–≤–æ–ª–æ–≤ 

Python —Ç–∞–∫–∂–µ –ø–æ–∑–≤–æ–ª—è–µ—Ç –ø–æ—Å–ª–µ `\` —É–∫–∞–∑—ã–≤–∞—Ç—å –≤–æ—Å—å–º–µ—Ä–∏—á–Ω–æ–µ –∏–ª–∏ —à–µ—Å—Ç–Ω–∞–¥—Ü–∞—Ç–µ—Ä–∏—á–Ω–æ–µ —á–∏—Å–ª–æ –¥–ª—è –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–∏—è —Å–∏–º–≤–æ–ª–∞. –ù–∞–ø—Ä–∏–º–µ—Ä, `\141` –∏ `\x61` –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è—é—Ç —Å—Ç—Ä–æ—á–Ω—É—é –±—É–∫–≤—É `a`, –ø–µ—Ä–≤–∞—è –≤ –≤–æ—Å—å–º–µ—Ä–∏—á–Ω–æ–º, –∞ –≤—Ç–æ—Ä–∞—è –≤ —à–µ—Å—Ç–Ω–∞–¥—Ü–∞—Ç–µ—Ä–∏—á–Ω–æ–º –≤–∏–¥–µ. –î—Ä—É–≥–æ–π —Å–ø–æ—Å–æ–± –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–∏—è —Å–∏–º–≤–æ–ª–æ–≤ - —Å–æ–ø—Ä–æ–≤–æ–∂–¥–µ–Ω–∏–µ `\u` –∫–æ–¥–æ–º —Å–∏–º–≤–æ–ª–æ–≤ Unicode, –Ω–∞–ø—Ä–∏–º–µ—Ä, `\u9a86\u660a` –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è–µ—Ç –∫–∏—Ç–∞–π—Å–∫–∏–π –∏–µ—Ä–æ–≥–ª–∏—Ñ "–õ–æ –•–∞–æ". –í—ã–ø–æ–ª–Ω–∏—Ç–µ —Å–ª–µ–¥—É—é—â–∏–π –∫–æ–¥ –∏ –ø–æ—Å–º–æ—Ç—Ä–∏—Ç–µ, —á—Ç–æ –ø–æ–ª—É—á–∏—Ç—Å—è –Ω–∞ –≤—ã—Ö–æ–¥–µ.

```python
s1 = '\141\142\143\x61\x62\x63'
s2 = '\u9a86\u660a'
print(s1)
print(s2)
```

### –°—Ç—Ä–æ–∫–æ–≤—ã–µ –æ–ø–µ—Ä–∞—Ü–∏–∏

Python –ø—Ä–µ–¥–æ—Å—Ç–∞–≤–ª—è–µ—Ç –æ—á–µ–Ω—å –±–æ–≥–∞—Ç—ã–π –Ω–∞–±–æ—Ä –æ–ø–µ—Ä–∞—Ç–æ—Ä–æ–≤ –¥–ª—è —Å—Ç—Ä–æ–∫–æ–≤—ã—Ö —Ç–∏–ø–æ–≤, –º–Ω–æ–≥–∏–µ –∏–∑ –∫–æ—Ç–æ—Ä—ã—Ö –∞–Ω–∞–ª–æ–≥–∏—á–Ω—ã –æ–ø–µ—Ä–∞—Ç–æ—Ä–∞–º –¥–ª—è —Å–ø–∏—Å–∫–æ–≤. –ù–∞–ø—Ä–∏–º–µ—Ä, –≤—ã –º–æ–∂–µ—Ç–µ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –æ–ø–µ—Ä–∞—Ç–æ—Ä `+` –¥–ª—è –æ–±—ä–µ–¥–∏–Ω–µ–Ω–∏—è —Å—Ç—Ä–æ–∫, –æ–ø–µ—Ä–∞—Ç–æ—Ä `*` –¥–ª—è –ø–æ–≤—Ç–æ—Ä–µ–Ω–∏—è —Å–æ–¥–µ—Ä–∂–∏–º–æ–≥–æ —Å—Ç—Ä–æ–∫–∏, –æ–ø–µ—Ä–∞—Ç–æ—Ä—ã `in` –∏ `not in` –¥–ª—è –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–∏—è —Ç–æ–≥–æ, —Å–æ–¥–µ—Ä–∂–∏—Ç –ª–∏ —Å—Ç—Ä–æ–∫–∞ –¥—Ä—É–≥—É—é —Å—Ç—Ä–æ–∫—É, –∞ —Ç–∞–∫–∂–µ –æ–ø–µ—Ä–∞—Ç–æ—Ä—ã `[]` –∏ `[:]` –¥–ª—è —É–¥–∞–ª–µ–Ω–∏—è —Å–∏–º–≤–æ–ª–∞ –∏–ª–∏ —Å–∏–º–≤–æ–ª–æ–≤ –∏–∑ —Å—Ç—Ä–æ–∫–∏.

#### –†–∞–∑–±–∏–µ–Ω–∏–µ –∏ –ø–æ–≤—Ç–æ—Ä–µ–Ω–∏–µ

–°–ª–µ–¥—É—é—â–∏–π –ø—Ä–∏–º–µ—Ä –¥–µ–º–æ–Ω—Å—Ç—Ä–∏—Ä—É–µ—Ç –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ –æ–ø–µ—Ä–∞—Ç–æ—Ä–æ–≤ `+` –∏ `*` –¥–ª—è –æ–ø–µ—Ä–∞—Ü–∏–π –æ–±—ä–µ–¥–∏–Ω–µ–Ω–∏—è –∏ –ø–æ–≤—Ç–æ—Ä–µ–Ω–∏—è —Å—Ç—Ä–æ–∫.

```python
s1 = 'hello' + ', ' + 'world'
print(s1)    # hello, world
s2 = '!' * 3
print(s2)    # !!!
s1 += s2
print(s1)    # hello, world!!!
s1 *= 2
print(s1)    # hello, world!!!hello, world!!!
```

–ü–æ–≤—Ç–æ—Ä–µ–Ω–∏–µ —Å—Ç—Ä–æ–∫ —Å –ø–æ–º–æ—â—å—é `*` - –æ—á–µ–Ω—å –∏–Ω—Ç–µ—Ä–µ—Å–Ω—ã–π –æ–ø–µ—Ä–∞—Ç–æ—Ä. –í–æ –º–Ω–æ–≥–∏—Ö —è–∑—ã–∫–∞—Ö –ø—Ä–æ–≥—Ä–∞–º–º–∏—Ä–æ–≤–∞–Ω–∏—è –¥–ª—è –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–∏—è —Å—Ç—Ä–æ–∫–∏ –∏–∑ 10 `a` –º–æ–∂–Ω–æ –Ω–∞–ø–∏—Å–∞—Ç—å —Ç–æ–ª—å–∫–æ `'aaaaaaaa`, –Ω–æ –≤ Python –º–æ–∂–Ω–æ –Ω–∞–ø–∏—Å–∞—Ç—å `'a' * 10`. –í—ã –º–æ–∂–µ—Ç–µ –ø–æ–¥—É–º–∞—Ç—å, —á—Ç–æ `'aaaaaaaa'` –Ω–µ —Ç–∞–∫ —É–∂ –∏ –Ω–µ—É–¥–æ–±–Ω–æ, –Ω–æ –ø–æ–¥—É–º–∞–π—Ç–µ, —á—Ç–æ –ø—Ä–æ–∏–∑–æ–π–¥–µ—Ç, –µ—Å–ª–∏ —Å–∏–º–≤–æ–ª `a` –±—É–¥–µ—Ç –ø–æ–≤—Ç–æ—Ä–µ–Ω 100 –∏–ª–∏ 1000 —Ä–∞–∑.

#### –û–ø–µ—Ä–∞—Ü–∏–∏ —Å—Ä–∞–≤–Ω–µ–Ω–∏—è

–î–ª—è –¥–≤—É—Ö –ø–µ—Ä–µ–º–µ–Ω–Ω—ã—Ö —Å—Ç—Ä–æ–∫–æ–≤–æ–≥–æ —Ç–∏–ø–∞ –º–æ–∂–Ω–æ –Ω–∞–ø—Ä—è–º—É—é –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –æ–ø–µ—Ä–∞—Ç–æ—Ä —Å—Ä–∞–≤–Ω–µ–Ω–∏—è, —á—Ç–æ–±—ã –æ–ø—Ä–µ–¥–µ–ª–∏—Ç—å —Ä–∞–≤–µ–Ω—Å—Ç–≤–æ –¥–≤—É—Ö —Å—Ç—Ä–æ–∫ –∏–ª–∏ —Å—Ä–∞–≤–Ω–∏—Ç—å –∏—Ö —Ä–∞–∑–º–µ—Ä—ã. –°–ª–µ–¥—É–µ—Ç –æ—Ç–º–µ—Ç–∏—Ç—å, —á—Ç–æ –ø–æ—Å–∫–æ–ª—å–∫—É —Å—Ç—Ä–æ–∫–∏ —Ç–∞–∫–∂–µ —Å—É—â–µ—Å—Ç–≤—É—é—Ç –≤ –¥–≤–æ–∏—á–Ω–æ–π —Ñ–æ—Ä–º–µ –≤ –ø–∞–º—è—Ç–∏ –∫–æ–º–ø—å—é—Ç–µ—Ä–∞, —Å—Ä–∞–≤–Ω–µ–Ω–∏–µ —Ä–∞–∑–º–µ—Ä–æ–≤ —Å—Ç—Ä–æ–∫ –æ—Å–Ω–æ–≤–∞–Ω–æ –Ω–∞ —Ä–∞–∑–º–µ—Ä–µ –∫–æ–¥–∏—Ä–æ–≤–∫–∏ –∫–∞–∂–¥–æ–≥–æ —Å–∏–º–≤–æ–ª–∞. –ù–∞–ø—Ä–∏–º–µ—Ä, –∫–æ–¥ `A` - `65`, –∞ –∫–æ–¥ `a` - `97`, –ø–æ—ç—Ç–æ–º—É —Ä–µ–∑—É–ª—å—Ç–∞—Ç `'A' < 'a'` –±—É–¥–µ—Ç —Ç–∞–∫–∏–º –∂–µ, –∫–∞–∫ —Ä–µ–∑—É–ª—å—Ç–∞—Ç `65 < 97`, —á—Ç–æ –æ—á–µ–≤–∏–¥–Ω–æ `True`; —Ç–æ–≥–¥–∞ –∫–∞–∫ —Ä–µ–∑—É–ª—å—Ç–∞—Ç `'boy' < 'bad'`, –ø–æ—Å–∫–æ–ª—å–∫—É –ø–µ—Ä–≤—ã–π —Å–∏–º–≤–æ–ª - `'b'`, –Ω–µ –º–æ–∂–µ—Ç –±—ã—Ç—å —Å—Ä–∞–≤–Ω–µ–Ω —Å —Ä–∞–∑–º–µ—Ä–æ–º –ø–µ—Ä–≤–æ–≥–æ —Å–∏–º–≤–æ–ª–∞, –ø–æ—ç—Ç–æ–º—É —Ñ–∞–∫—Ç–∏—á–µ—Å–∫–æ–µ —Å—Ä–∞–≤–Ω–µ–Ω–∏–µ - —ç—Ç–æ —Ä–∞–∑–º–µ—Ä –≤—Ç–æ—Ä–æ–≥–æ —Å–∏–º–≤–æ–ª–∞, –∏ –æ—á–µ–≤–∏–¥–Ω–æ, —á—Ç–æ `'o' < ' a'` –ø—Ä–∏–≤–æ–¥–∏—Ç –∫ `False`, —Ç–∞–∫ —á—Ç–æ `'boy' < 'bad'` –ø—Ä–∏–≤–æ–¥–∏—Ç –∫ `False`. –ï—Å–ª–∏ –Ω–µ—è—Å–Ω–æ, –∫–∞–∫–∞—è –∏–º–µ–Ω–Ω–æ –∫–æ–¥–∏—Ä–æ–≤–∫–∞ —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤—É–µ—Ç –¥–≤—É–º —Å–∏–º–≤–æ–ª–∞–º, –≤—ã –º–æ–∂–µ—Ç–µ –ø–æ–ª—É—á–∏—Ç—å –µ–µ —Å –ø–æ–º–æ—â—å—é —Ñ—É–Ω–∫—Ü–∏–∏ `ord`, –æ –∫–æ—Ç–æ—Ä–æ–π –º—ã —É–∂–µ –≥–æ–≤–æ—Ä–∏–ª–∏. –ù–∞–ø—Ä–∏–º–µ—Ä, –∑–Ω–∞—á–µ–Ω–∏–µ `ord('A')` —Ä–∞–≤–Ω–æ `65`, –∞ –∑–Ω–∞—á–µ–Ω–∏–µ `ord('hao')` —Ä–∞–≤–Ω–æ `26122`. –°–ª–µ–¥—É—é—â–∏–π –∫–æ–¥ –¥–µ–º–æ–Ω—Å—Ç—Ä–∏—Ä—É–µ—Ç –æ–ø–µ—Ä–∞—Ü–∏—é —Å—Ä–∞–≤–Ω–µ–Ω–∏—è —Å—Ç—Ä–æ–∫, –ø–æ—ç—Ç–æ–º—É —Ä–∞—Å—Å–º–æ—Ç—Ä–∏—Ç–µ –µ–µ –ø–æ–¥—Ä–æ–±–Ω–µ–µ.

```python
s1 = 'a whole new world'
s2 = 'hello world'
print(s1 == s2)             # False
print(s1 < s2)              # True
print(s1 == 'hello world')  # False
print(s2 == 'hello world')  # True
print(s2 != 'Hello world')  # True
s3 = '–õ–æ'
print(ord('–õ'))            # 1051
print(ord('–æ'))            # 1086
s4 = '–•–∞–æ'
print(ord('–•'))            # 1061
print(ord('–∞'))            # 1072
print(ord('–æ'))            # 1086
print(s3 >= s4)             # False
print(s3 != s4)             # True
```

#### –û–ø–µ—Ä–∞—Ç–æ—Ä—ã –ø—Ä–∏–Ω–∞–¥–ª–µ–∂–Ω–æ—Å—Ç–∏

Python –º–æ–∂–µ—Ç –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å `in` –∏ `not in` –¥–ª—è –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–∏—è —Ç–æ–≥–æ, —Å–æ–¥–µ—Ä–∂–∏—Ç –ª–∏ —Å—Ç—Ä–æ–∫–∞ –¥—Ä—É–≥–æ–π —Å–∏–º–≤–æ–ª –∏–ª–∏ —Å—Ç—Ä–æ–∫—É. –ö–∞–∫ –∏ –¥–ª—è —Å–ø–∏—Å–∫–æ–≤, `in` –∏ `not in` –Ω–∞–∑—ã–≤–∞—é—Ç—Å—è –æ–ø–µ—Ä–∞—Ç–æ—Ä–∞–º–∏ –ø—Ä–∏–Ω–∞–¥–ª–µ–∂–Ω–æ—Å—Ç–∏ –∏ –¥–∞—é—Ç –±—É–ª–µ–≤–æ –∑–Ω–∞—á–µ–Ω–∏–µ `True` –∏–ª–∏ `False`, –∫–∞–∫ –ø–æ–∫–∞–∑–∞–Ω–æ –≤ –∫–æ–¥–µ –Ω–∏–∂–µ.

```python
s1 = 'hello, world'
s2 = 'goodbye, world'
print('wo' in s1)      # True
print('wo' not in s2)  # False
print(s2 in s1)        # False
```

#### –ü–æ–ª—É—á–µ–Ω–∏–µ –¥–ª–∏–Ω—ã —Å—Ç—Ä–æ–∫–∏

–ü–æ–ª—É—á–∏—Ç—å –¥–ª–∏–Ω—É —Å—Ç—Ä–æ–∫–∏ - —Ç–æ –∂–µ —Å–∞–º–æ–µ, —á—Ç–æ –ø–æ–ª—É—á–∏—Ç—å –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ —ç–ª–µ–º–µ–Ω—Ç–æ–≤ –≤ —Å–ø–∏—Å–∫–µ, –∏—Å–ø–æ–ª—å–∑—É—è –≤—Å—Ç—Ä–æ–µ–Ω–Ω—É—é —Ñ—É–Ω–∫—Ü–∏—é `len`, –∫–∞–∫ –ø–æ–∫–∞–∑–∞–Ω–æ –≤ —Å–ª–µ–¥—É—é—â–µ–º –∫–æ–¥–µ.

```python
s = 'hello, world'
print(len(s))                 # 12
print(len('goodbye, world'))  # 14
```

#### –ò–Ω–¥–µ–∫—Å–∏—Ä–æ–≤–∞–Ω–∏–µ –∏ —Å—Ä–µ–∑—ã

–û–ø–µ—Ä–∞—Ü–∏–∏ –∏–Ω–¥–µ–∫—Å–∏—Ä–æ–≤–∞–Ω–∏—è –∏ –Ω–∞—Ä–µ–∑–∫–∏ —Å—Ç—Ä–æ–∫ –ø—Ä–∞–∫—Ç–∏—á–µ—Å–∫–∏ –Ω–µ–æ—Ç–ª–∏—á–∏–º—ã –æ—Ç –æ–ø–µ—Ä–∞—Ü–∏–π —Å–æ —Å–ø–∏—Å–∫–∞–º–∏ –∏ –∫–æ—Ä—Ç–µ–∂–∞–º–∏, –ø–æ—Å–∫–æ–ª—å–∫—É —Å—Ç—Ä–æ–∫–∞ - —ç—Ç–æ —Ç–æ–∂–µ —É–ø–æ—Ä—è–¥–æ—á–µ–Ω–Ω–∞—è –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç—å, –∫ —ç–ª–µ–º–µ–Ω—Ç–∞–º –∫–æ—Ç–æ—Ä–æ–π –º–æ–∂–Ω–æ –æ–±—Ä–∞—â–∞—Ç—å—Å—è —Å –ø–æ–º–æ—â—å—é –ø—Ä—è–º–æ–π –∏–ª–∏ –æ–±—Ä–∞—Ç–Ω–æ–π —Ü–µ–ª–æ—á–∏—Å–ª–µ–Ω–Ω–æ–π –∏–Ω–¥–µ–∫—Å–∞—Ü–∏–∏. –û–¥–Ω–∞–∫–æ —Å–ª–µ–¥—É–µ—Ç –æ—Ç–º–µ—Ç–∏—Ç—å, —á—Ç–æ –ø–æ—Å–∫–æ–ª—å–∫—É **—Å—Ç—Ä–æ–∫–∏ —è–≤–ª—è—é—Ç—Å—è –Ω–µ–∏–∑–º–µ–Ω—è–µ–º—ã–º–∏ —Ç–∏–ø–∞–º–∏**, **–≤—ã –Ω–µ –º–æ–∂–µ—Ç–µ –∏–∑–º–µ–Ω—è—Ç—å —Å–∏–º–≤–æ–ª—ã –≤ —Å—Ç—Ä–æ–∫–µ —Å –ø–æ–º–æ—â—å—é –æ–ø–µ—Ä–∞—Ü–∏–π –∏–Ω–¥–µ–∫—Å–∏—Ä–æ–≤–∞–Ω–∏—è**.

```python
s = 'abc123456'
n = len(s)
print(s[0], s[-n])    # a a
print(s[n-1], s[-1])  # 6 6
print(s[2], s[-7])    # c c
print(s[5], s[-4])    # 3 3
print(s[2:5])         # c12
print(s[-7:-4])       # c12
print(s[2:])          # c123456
print(s[:2])          # ab
print(s[::2])         # ac246
print(s[::-1])        # 654321cba
```

–í–∞–∂–Ω–æ –µ—â–µ —Ä–∞–∑ –æ—Ç–º–µ—Ç–∏—Ç—å, —á—Ç–æ –ø—Ä–∏ –≤—ã–ø–æ–ª–Ω–µ–Ω–∏–∏ –æ–ø–µ—Ä–∞—Ü–∏–π –∏–Ω–¥–µ–∫—Å–∏—Ä–æ–≤–∞–Ω–∏—è, –µ—Å–ª–∏ –∏–Ω–¥–µ–∫—Å –≤—ã—Ö–æ–¥–∏—Ç –∑–∞ –ø—Ä–µ–¥–µ–ª—ã –¥–∏–∞–ø–∞–∑–æ–Ω–∞, –±—É–¥–µ—Ç –≤—ã–¥–∞–Ω–æ –∏—Å–∫–ª—é—á–µ–Ω–∏–µ `IndexError` —Å —Å–æ–æ–±—â–µ–Ω–∏–µ–º –æ–± –æ—à–∏–±–∫–µ `string index out of range`.

### –ü–µ—Ä–µ–±–æ—Ä —Å–∏–º–≤–æ–ª–æ–≤

–ï—Å–ª–∏ –≤—ã —Ö–æ—Ç–∏—Ç–µ –ø–µ—Ä–µ–±—Ä–∞—Ç—å –∫–∞–∂–¥—ã–π —Å–∏–º–≤–æ–ª –≤ —Å—Ç—Ä–æ–∫–µ, —Ç–æ –º–æ–∂–µ—Ç–µ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å —Ü–∏–∫–ª `for-in` –æ–¥–Ω–∏–º –∏–∑ –¥–≤—É—Ö —Å–ø–æ—Å–æ–±–æ–≤, –∫–∞–∫ –ø–æ–∫–∞–∑–∞–Ω–æ –Ω–∏–∂–µ.

–ü–µ—Ä–≤—ã–π —Å–ø–æ—Å–æ–±:

```python
s = 'hello'
for i in range(len(s)):
    print(s[i])
```

–í—Ç–æ—Ä–æ–π —Å–ø–æ—Å–æ–±:

```python
s = 'hello'
for elem in s:
    print(elem)
```

### –°—Ç—Ä–æ–∫–æ–≤—ã–µ –º–µ—Ç–æ–¥—ã

–í Python –º—ã –º–æ–∂–µ–º –º–∞–Ω–∏–ø—É–ª–∏—Ä–æ–≤–∞—Ç—å –∏ —Ä–∞–±–æ—Ç–∞—Ç—å —Å–æ —Å—Ç—Ä–æ–∫–∞–º–∏ —Å –ø–æ–º–æ—â—å—é –º–µ—Ç–æ–¥–æ–≤, –∫–æ—Ç–æ—Ä—ã–µ –ø–æ—Å—Ç–∞–≤–ª—è—é—Ç—Å—è —Å —Ç–∏–ø–æ–º string. –ü—Ä–µ–¥–ø–æ–ª–æ–∂–∏–º, —á—Ç–æ —É –Ω–∞—Å –µ—Å—Ç—å —Å—Ç—Ä–æ–∫–∞ —Å –∏–º–µ–Ω–µ–º `foo` –∏ —É –Ω–µ—ë –µ—Å—Ç—å –º–µ—Ç–æ–¥ —Å –∏–º–µ–Ω–µ–º `bar`, —Ç–æ–≥–¥–∞ —Å–∏–Ω—Ç–∞–∫—Å–∏—Å –¥–ª—è –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è –º–µ—Ç–æ–¥–∞ –¥–∞–Ω–Ω–æ–π —Å—Ç—Ä–æ–∫–∏ –±—É–¥–µ—Ç `foo.bar()`, —á—Ç–æ —è–≤–ª—è–µ—Ç—Å—è —Å–∏–Ω—Ç–∞–∫—Å–∏—Å–æ–º –¥–ª—è –≤—ã–∑–æ–≤–∞ –º–µ—Ç–æ–¥–∞ –æ–±—ä–µ–∫—Ç–∞ –ø–æ —Å—Å—ã–ª–∫–µ –Ω–∞ –æ–±—ä–µ–∫—Ç, —Ç–∞–∫–∏–º –∂–µ, –∫–∞–∫ —Å–∏–Ω—Ç–∞–∫—Å–∏—Å –¥–ª—è –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è –º–µ—Ç–æ–¥–∞ —Å–ø–∏—Å–∫–∞, –∫–∞–∫ –æ–ø–∏—Å–∞–Ω–æ —Ä–∞–Ω–µ–µ. –≠—Ç–æ —Ç–∞–∫–æ–π –∂–µ —Å–∏–Ω—Ç–∞–∫—Å–∏—Å –≤—ã–∑–æ–≤–∞ –º–µ—Ç–æ–¥–∞ –æ–±—ä–µ–∫—Ç–∞ –ø–æ —Å—Å—ã–ª–∫–µ –Ω–∞ –æ–±—ä–µ–∫—Ç, –∫–∞–∫ –∏ –º–µ—Ç–æ–¥ —Å–ø–∏—Å–∫–∞, –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–Ω—ã–π —Ä–∞–Ω–µ–µ.

#### –û–ø–µ—Ä–∞—Ü–∏–∏, —Å–≤—è–∑–∞–Ω–Ω—ã–µ —Å —Ä–µ–≥–∏—Å—Ç—Ä–æ–º

–°–ª–µ–¥—É—é—â–∏–π –∫–æ–¥ –¥–µ–º–æ–Ω—Å—Ç—Ä–∏—Ä—É–µ—Ç –º–µ—Ç–æ–¥—ã, —Å–≤—è–∑–∞–Ω–Ω—ã–µ —Å –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏–µ–º —Ä–µ–≥–∏—Å—Ç—Ä–∞ —Å—Ç—Ä–æ–∫.

```python
s1 = 'hello, world!'
# –ü–µ—Ä–µ–≤–µ—Å—Ç–∏ –ø–µ—Ä–≤—É—é –±—É–∫–≤—É —Å—Ç—Ä–æ–∫–∏ –≤ –≤–µ—Ä—Ö–Ω–∏–π —Ä–µ–≥–∏—Å—Ç—Ä
print(s1.capitalize())  # Hello, world!
# –ó–∞–ø–∏—Å–∞—Ç—å –ø–µ—Ä–≤—É—é –±—É–∫–≤—É –∫–∞–∂–¥–æ–≥–æ —Å–ª–æ–≤–∞ –≤ –∑–∞–≥–ª–∞–≤–Ω—É—é
print(s1.title())       # Hello, World!
# –°—Ç—Ä–æ–∫–∞ –≤ –≤–µ—Ä—Ö–Ω–µ–º —Ä–µ–≥–∏—Å—Ç—Ä–µ
print(s1.upper())       # HELLO, WORLD!
s2 = 'GOODBYE'
# –°—Ç—Ä–æ–∫–∞ –≤ –Ω–∏–∂–Ω–µ–º —Ä–µ–≥–∏—Å—Ç—Ä–µ
print(s2.lower())       # goodbye
# –ü—Ä–æ–≤–µ—Ä–∏—Ç—å –∑–Ω–∞—á–µ–Ω–∏—è s1 –∏ s2
print(s1)               # hello, world
print(s2)               # GOODBYE
```

> **–ü—Ä–∏–º–µ—á–∞–Ω–∏–µ**: –ü–æ—Å–∫–æ–ª—å–∫—É —Å—Ç—Ä–æ–∫–∞ —è–≤–ª—è–µ—Ç—Å—è –Ω–µ–∏–∑–º–µ–Ω—è–µ–º—ã–º —Ç–∏–ø–æ–º, –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ —Å—Ç—Ä–æ–∫–æ–≤—ã—Ö –º–µ—Ç–æ–¥–æ–≤ –¥–ª—è —Ä–∞–±–æ—Ç—ã —Å–æ —Å—Ç—Ä–æ–∫–∞–º–∏ –ø—Ä–∏–≤–æ–¥–∏—Ç –∫ —Å–æ–∑–¥–∞–Ω–∏—é –Ω–æ–≤—ã—Ö —Å—Ç—Ä–æ–∫, –Ω–æ –∑–Ω–∞—á–µ–Ω–∏—è –∏—Å—Ö–æ–¥–Ω—ã—Ö –ø–µ—Ä–µ–º–µ–Ω–Ω—ã—Ö –ø—Ä–∏ —ç—Ç–æ–º –Ω–µ –∏–∑–º–µ–Ω—è–µ—Ç—Å—è. –ü–æ—ç—Ç–æ–º—É –≤ –ø—Ä–∏–≤–µ–¥–µ–Ω–Ω–æ–º –≤—ã—à–µ –∫–æ–¥–µ, –∫–æ–≥–¥–∞ –º—ã –≤ –∫–æ–Ω—Ü–µ –ø—Ä–æ–≤–µ—Ä—è–µ–º –∑–Ω–∞—á–µ–Ω–∏—è –ø–µ—Ä–µ–º–µ–Ω–Ω—ã—Ö `s1` –∏ `s2`, –∏—Ö –∑–Ω–∞—á–µ–Ω–∏—è –Ω–µ –º–µ–Ω—è—é—Ç—Å—è.

#### –û–ø–µ—Ä–∞—Ü–∏–∏ –ø–æ–∏—Å–∫–∞

–ï—Å–ª–∏ –≤—ã —Ö–æ—Ç–∏—Ç–µ —É–∑–Ω–∞—Ç—å, –µ—Å—Ç—å –ª–∏ –≤ —Å—Ç—Ä–æ–∫–µ –¥—Ä—É–≥–∞—è —Å—Ç—Ä–æ–∫–∞, —Ä–∞—Å–ø–æ–ª–æ–∂–µ–Ω–Ω–∞—è –≤–Ω—É—Ç—Ä–∏ –Ω–µ—ë, —Ç–æ –º–æ–∂–µ—Ç–µ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –º–µ—Ç–æ–¥—ã `find` –∏–ª–∏ `index` –¥–ª—è —Å—Ç—Ä–æ–∫. –ü—Ä–∏ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–∏ –º–µ—Ç–æ–¥–æ–≤ `find` –∏ `index` –≤—ã —Ç–∞–∫–∂–µ –º–æ–∂–µ—Ç–µ —É–∫–∞–∑–∞—Ç—å –æ–±–ª–∞—Å—Ç—å –ø–æ–∏—Å–∫–∞ —Å –ø–æ–º–æ—â—å—é –∞—Ä–≥—É–º–µ–Ω—Ç–æ–≤ –º–µ—Ç–æ–¥–∞, —Ç–æ –µ—Å—Ç—å –ø–æ–∏—Å–∫ –Ω–µ –æ–±—è–∑–∞—Ç–µ–ª—å–Ω–æ –¥–æ–ª–∂–µ–Ω –Ω–∞—á–∏–Ω–∞—Ç—å—Å—è —Å –∏–Ω–¥–µ–∫—Å–∞ `0`.

```python
s = 'hello, world!'

print(s.find('or', 9))   # -1
print(s.find('of'))      # -1
print(s.index('or'))     # 8
print(s.index('or', 9))  # ValueError: substring not found
```

>**–ü—Ä–∏–º–µ—á–∞–Ω–∏–µ**: –ú–µ—Ç–æ–¥ `find` –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç `-1`, –µ—Å–ª–∏ –Ω–µ –Ω–∞—Ö–æ–¥–∏—Ç —É–∫–∞–∑–∞–Ω–Ω—É—é –ø–æ–¥—Å—Ç—Ä–æ–∫—É, –∞ –º–µ—Ç–æ–¥ `index` –≤—ã–∑—ã–≤–∞–µ—Ç –æ—à–∏–±–∫—É `ValueError`, –µ—Å–ª–∏ –Ω–µ –Ω–∞—Ö–æ–¥–∏—Ç —É–∫–∞–∑–∞–Ω–Ω—É—é —Å—Ç—Ä–æ–∫—É.

–ú–µ—Ç–æ–¥—ã `find` –∏ `index` —Ç–∞–∫–∂–µ –∏–º–µ—é—Ç –≤–µ—Ä—Å–∏–∏ —Å –æ–±—Ä–∞—Ç–Ω—ã–º –ø–æ–∏—Å–∫–æ–º (back-to-front), `rfind` –∏ `rindex`, —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤–µ–Ω–Ω–æ, –∫–∞–∫ –ø–æ–∫–∞–∑–∞–Ω–æ –≤ –∫–æ–¥–µ –Ω–∏–∂–µ.

```python
s = 'hello world!'

# print(s.rindex('o', 8))  # ValueError: substring not found
```

#### –°—É–∂–¥–µ–Ω–∏–µ –æ —Ö–∞—Ä–∞–∫—Ç–µ—Ä–µ

–í—ã –º–æ–∂–µ—Ç–µ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å `startswith` –∏ `endswith`, —á—Ç–æ–±—ã –æ–ø—Ä–µ–¥–µ–ª–∏—Ç—å, –Ω–∞—á–∏–Ω–∞–µ—Ç—Å—è –∏ –∑–∞–∫–∞–Ω—á–∏–≤–∞–µ—Ç—Å—è –ª–∏ —Å—Ç—Ä–æ–∫–∞ —Å –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–Ω–æ–π –ø–æ–¥—Å—Ç—Ä–æ–∫–∏; –≤—ã —Ç–∞–∫–∂–µ –º–æ–∂–µ—Ç–µ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –º–µ—Ç–æ–¥—ã, –Ω–∞—á–∏–Ω–∞—é—â–∏–µ—Å—è —Å `is`, —á—Ç–æ–±—ã –æ–ø—Ä–µ–¥–µ–ª–∏—Ç—å —Ö–∞—Ä–∞–∫—Ç–µ—Ä–∏—Å—Ç–∏–∫–∏ —Å—Ç—Ä–æ–∫–∏, –∏ –≤—Å–µ —ç—Ç–∏ –º–µ—Ç–æ–¥—ã –≤–æ–∑–≤—Ä–∞—â–∞—é—Ç –±—É–ª–µ–≤—ã –∑–Ω–∞—á–µ–Ω–∏—è, –∫–∞–∫ –ø–æ–∫–∞–∑–∞–Ω–æ –≤ —Å–ª–µ–¥—É—é—â–µ–º –∫–æ–¥–µ.

> **–ü—Ä–∏–º–µ—á–∞–Ω–∏–µ**: –í—ã—à–µ—É–∫–∞–∑–∞–Ω–Ω—ã–µ `isdigit` –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –¥–ª—è –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–∏—è —Ç–æ–≥–æ, —Å–æ—Å—Ç–æ–∏—Ç –ª–∏ —Å—Ç—Ä–æ–∫–∞ –ø–æ–ª–Ω–æ—Å—Ç—å—é –∏–∑ —Ü–∏—Ñ—Ä, `isalpha` –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –¥–ª—è –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–∏—è —Ç–æ–≥–æ, —Å–æ—Å—Ç–æ–∏—Ç –ª–∏ —Å—Ç—Ä–æ–∫–∞ –ø–æ–ª–Ω–æ—Å—Ç—å—é –∏–∑ –±—É–∫–≤, –≥–¥–µ –±—É–∫–≤—ã –æ–∑–Ω–∞—á–∞—é—Ç —Å–∏–º–≤–æ–ª—ã Unicode, –Ω–æ –Ω–µ —Å–∏–º–≤–æ–ª—ã Emoji, –∏ `isalnum` –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –¥–ª—è –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–∏—è —Ç–æ–≥–æ, —Å–æ—Å—Ç–æ–∏—Ç –ª–∏ —Å—Ç—Ä–æ–∫–∞ –∫–∞–∫ –∏–∑ –±—É–∫–≤, —Ç–∞–∫ –∏ –∏–∑ —Ü–∏—Ñ—Ä.

#### –§–æ—Ä–º–∞—Ç–∏—Ä–æ–≤–∞–Ω–∏–µ

–í Python —Ç–∏–ø—ã —Å—Ç—Ä–æ–∫ –º–æ–∂–Ω–æ —Ü–µ–Ω—Ç—Ä–∏—Ä–æ–≤–∞—Ç—å, –≤—ã—Ä–∞–≤–Ω–∏–≤–∞—Ç—å –ø–æ –ª–µ–≤–æ–º—É –∫—Ä–∞—é –∏ –ø—Ä–∞–≤–æ–º—É —Å –ø–æ–º–æ—â—å—é –º–µ—Ç–æ–¥–æ–≤ `centre`, `ljust` –∏ `rjust`. –ú–µ—Ç–æ–¥ `zfill` —Ç–∞–∫–∂–µ –º–æ–∂–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å, –µ—Å–ª–∏ –≤—ã —Ö–æ—Ç–∏—Ç–µ –≤—Å—Ç–∞–≤–∏—Ç—å –Ω—É–ª–∏ –≤ –ª–µ–≤—É—é —á–∞—Å—Ç—å —Å—Ç—Ä–æ–∫–∏.

#### –ó–∞–º–µ–Ω–∞

–ï—Å–ª–∏ –≤—ã —Ö–æ—Ç–∏—Ç–µ –∑–∞–º–µ–Ω–∏—Ç—å —Å–æ–¥–µ—Ä–∂–∏–º–æ–µ, —É–∫–∞–∑–∞–Ω–Ω–æ–µ –≤ —Å—Ç—Ä–æ–∫–µ, –Ω–æ–≤—ã–º —Å–æ–¥–µ—Ä–∂–∏–º—ã–º, —Ç–æ –º–æ–∂–µ—Ç–µ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –º–µ—Ç–æ–¥ `replace`, –∫–∞–∫ –ø–æ–∫–∞–∑–∞–Ω–æ –≤ –∫–æ–¥–µ –Ω–∏–∂–µ. –ü–µ—Ä–≤—ã–º –ø–∞—Ä–∞–º–µ—Ç—Ä–æ–º –º–µ—Ç–æ–¥–∞ `replace` —è–≤–ª—è–µ—Ç—Å—è —Å–æ–¥–µ—Ä–∂–∏–º–æ–µ, –∫–æ—Ç–æ—Ä–æ–µ –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ –∑–∞–º–µ–Ω–∏—Ç—å, –≤—Ç–æ—Ä—ã–º - –∑–∞–º–µ–Ω—è–µ–º–æ–µ —Å–æ–¥–µ—Ä–∂–∏–º–æ–µ, –∞ —Ç—Ä–µ—Ç–∏–π –ø–∞—Ä–∞–º–µ—Ç—Ä –ø–æ–∑–≤–æ–ª—è–µ—Ç —É–∫–∞–∑–∞—Ç—å –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ —Ä–∞–∑, –∫–æ—Ç–æ—Ä–æ–µ –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ –∑–∞–º–µ–Ω–∏—Ç—å —Å–æ–¥–µ—Ä–∂–∏–º–æ–µ.

```python
s = 'hello, good world'
print(s.replace('o', '@'))     # hell@, g@@d w@rld
print(s.replace('o', '@', 1))  # hell@, good world
```

#### –†–∞–∑–¥–µ–ª–µ–Ω–∏–µ –∏ –æ–±—ä–µ–¥–∏–Ω–µ–Ω–∏–µ

–í—ã –º–æ–∂–µ—Ç–µ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å —Å—Ç—Ä–æ–∫–æ–≤—ã–π –º–µ—Ç–æ–¥ `split` –¥–ª—è —Ä–∞–∑–¥–µ–ª–µ–Ω–∏—è —Å—Ç—Ä–æ–∫–∏ –Ω–∞ –Ω–µ—Å–∫–æ–ª—å–∫–æ —Å—Ç—Ä–æ–∫ (–≤ —Å–ø–∏—Å–∫–µ), –∏–ª–∏ –º–µ—Ç–æ–¥ `join` –¥–ª—è –æ–±—ä–µ–¥–∏–Ω–µ–Ω–∏—è –Ω–µ—Å–∫–æ–ª—å–∫–∏—Ö —Å—Ç—Ä–æ–∫ –≤ —Å–ø–∏—Å–∫–µ –≤ –æ–¥–Ω—É —Å—Ç—Ä–æ–∫—É, –∫–∞–∫ –ø–æ–∫–∞–∑–∞–Ω–æ –≤ –∫–æ–¥–µ –Ω–∏–∂–µ.

```python
s = 'I love you'
words = s.split()
print(words)            # ['I', 'love', 'you']
print('~'.join(words))  # I~love~you
```

–°–ª–µ–¥—É–µ—Ç –æ—Ç–º–µ—Ç–∏—Ç—å, —á—Ç–æ –º–µ—Ç–æ–¥ `split` –ø–æ —É–º–æ–ª—á–∞–Ω–∏—é –∏—Å–ø–æ–ª—å–∑—É–µ—Ç –¥–ª—è —Ä–∞–∑–±–∏–µ–Ω–∏—è –ø—Ä–æ–±–µ–ª—ã, –º—ã —Ç–∞–∫–∂–µ –º–æ–∂–µ–º —É–∫–∞–∑–∞—Ç—å –¥—Ä—É–≥–∏–µ —Å–∏–º–≤–æ–ª—ã –¥–ª—è —Ä–∞–∑–±–∏–µ–Ω–∏—è —Å—Ç—Ä–æ–∫–∏, –∏ –≤—ã —Ç–∞–∫–∂–µ –º–æ–∂–µ—Ç–µ —É–∫–∞–∑–∞—Ç—å –º–∞–∫—Å–∏–º–∞–ª—å–Ω–æ–µ –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ —Ä–∞–∑–±–∏–µ–Ω–∏–π, —á—Ç–æ–±—ã –∫–æ–Ω—Ç—Ä–æ–ª–∏—Ä–æ–≤–∞—Ç—å —ç—Ñ—Ñ–µ–∫—Ç —Ä–∞–∑–±–∏–µ–Ω–∏—è, –∫–æ–¥ –ø–æ–∫–∞–∑–∞–Ω –Ω–∏–∂–µ.

```python
s = 'I#love#you#so#much'
words = s.split('#')
print(words)  # ['I', 'love', 'you', 'so', 'much']
words = s.split('#', 2)
print(words)  # ['I', 'love', 'you#so#much']
```

#### –ö–æ–¥–∏—Ä–æ–≤–∞–Ω–∏–µ –∏ –¥–µ–∫–æ–¥–∏—Ä–æ–≤–∞–Ω–∏–µ

–í –¥–æ–ø–æ–ª–Ω–µ–Ω–∏–µ –∫ —Å—Ç—Ä–æ–∫–æ–≤–æ–º—É —Ç–∏–ø—É `str` –≤ Python –µ—Å—Ç—å —Ç–∏–ø –±–∞–π—Ç–æ–≤–æ–π —Å—Ç—Ä–æ–∫–∏ (`bytes`) –¥–ª—è –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–∏—è –¥–≤–æ–∏—á–Ω—ã—Ö –¥–∞–Ω–Ω—ã—Ö. –ë–∞–π—Ç–æ–≤–∞—è —Å—Ç—Ä–æ–∫–∞ - —ç—Ç–æ **–∫–æ–Ω–µ—á–Ω–∞—è –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç—å –∏–∑ –Ω—É–ª—è –∏–ª–∏ –±–æ–ª–µ–µ –±–∞–π—Ç–æ–≤**. –° –ø–æ–º–æ—â—å—é —Å—Ç—Ä–æ–∫–æ–≤–æ–≥–æ –º–µ—Ç–æ–¥–∞ `encode` —Å—Ç—Ä–æ–∫–∏ –º—ã –º–æ–∂–µ–º –∑–∞–∫–æ–¥–∏—Ä–æ–≤–∞—Ç—å —Å—Ç—Ä–æ–∫—É –≤ –±–∞–π—Ç–æ–≤—É—é —Å—Ç—Ä–æ–∫—É –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–Ω—ã–º —Å–ø–æ—Å–æ–±–æ–º –∫–æ–¥–∏—Ä–æ–≤–∞–Ω–∏—è, –∞ —Ç–∞–∫–∂–µ –¥–µ–∫–æ–¥–∏—Ä–æ–≤–∞—Ç—å –±–∞–π—Ç–æ–≤—É—é —Å—Ç—Ä–æ–∫—É –≤ —Å—Ç—Ä–æ–∫—É —Å –ø–æ–º–æ—â—å—é –º–µ—Ç–æ–¥–∞ –±–∞–π—Ç–æ–≤–æ–π —Å—Ç—Ä–æ–∫–∏ `decode`, –∫–∞–∫ –ø–æ–∫–∞–∑–∞–Ω–æ –≤ —Å–ª–µ–¥—É—é—â–µ–º –∫–æ–¥–µ.

```python
a = '–õ–æ –•–∞–æ'
b = a.encode('utf-8')
c = a.encode('gbk')
print(b)                  # b'\xe9\xaa\x86\xe6\x98\x8a'
print(c)                  # b'\xc2\xe6\xea\xbb'
print(b.decode('utf-8'))  # –õ–æ –•–∞–æ
print(c.decode('gbk'))    # –õ–æ –•–∞–æ
```

–û–±—Ä–∞—Ç–∏—Ç–µ –≤–Ω–∏–º–∞–Ω–∏–µ, —á—Ç–æ –Ω–µ—Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤–∏—è –≤ –∫–æ–¥–∏—Ä–æ–≤–∞–Ω–∏–∏ –∏ –¥–µ–∫–æ–¥–∏—Ä–æ–≤–∞–Ω–∏–∏ –º–æ–≥—É—Ç –ø—Ä–∏–≤–µ—Å—Ç–∏ –∫ –ø–æ—è–≤–ª–µ–Ω–∏—é –æ—à–∏–±–æ–∫ (–Ω–µ–≤–æ–∑–º–æ–∂–Ω–æ—Å—Ç—å –≤–æ—Å–ø—Ä–æ–∏–∑–≤–µ—Å—Ç–∏ –æ—Ä–∏–≥–∏–Ω–∞–ª—å–Ω–æ–µ —Å–æ–¥–µ—Ä–∂–∏–º–æ–µ) –∏–ª–∏ –≤—ã–∑–≤–∞—Ç—å –æ—à–∏–±–∫—É `UnicodeDecodeError`, —á—Ç–æ –º–æ–∂–µ—Ç –ø—Ä–∏–≤–µ—Å—Ç–∏ –∫ –∞–≤–∞—Ä–∏–π–Ω–æ–º—É –∑–∞–≤–µ—Ä—à–µ–Ω–∏—é —Ä–∞–±–æ—Ç—ã –ø—Ä–æ–≥—Ä–∞–º–º—ã.

#### –î—Ä—É–≥–∏–µ –º–µ—Ç–æ–¥—ã

–ï—â–µ –æ–¥–Ω–æ–π —Ä–∞—Å–ø—Ä–æ—Å—Ç—Ä–∞–Ω–µ–Ω–Ω–æ–π –æ–ø–µ—Ä–∞—Ü–∏–µ–π –¥–ª—è —Å—Ç—Ä–æ–∫–æ–≤–æ–≥–æ —Ç–∏–ø–∞ —è–≤–ª—è–µ—Ç—Å—è –ø—Ä–æ–≤–µ—Ä–∫–∞ —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤–∏—è —Å—Ç—Ä–æ–∫–∏, —Ç.–µ. –ø—Ä–æ–≤–µ—Ä–∫–∞ —Ç–æ–≥–æ, —É–¥–æ–≤–ª–µ—Ç–≤–æ—Ä—è–µ—Ç –ª–∏ —Å—Ç—Ä–æ–∫–∞ –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–Ω–æ–º—É —à–∞–±–ª–æ–Ω—É. –ù–∞–ø—Ä–∏–º–µ—Ä, –ø—Ä–æ–≤–µ—Ä–∫–∞ –Ω–∞ —Å–∞–π—Ç–µ –∏–º–µ–Ω –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª–µ–π –∏ –∞–¥—Ä–µ—Å–æ–≤ —ç–ª–µ–∫—Ç—Ä–æ–Ω–Ω–æ–π –ø–æ—á—Ç—ã –≤ —Ä–µ–≥–∏—Å—Ç—Ä–∞—Ü–∏–æ–Ω–Ω–æ–π –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏–∏ –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—è - —ç—Ç–æ –ø—Ä–æ–≤–µ—Ä–∫–∞ –Ω–∞ —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤–∏–µ —à–∞–±–ª–æ–Ω—É. –ò–Ω—Å—Ç—Ä—É–º–µ–Ω—Ç, –æ–±–µ—Å–ø–µ—á–∏–≤–∞—é—â–∏–π –ø—Ä–æ–≤–µ—Ä–∫—É –ø–æ —à–∞–±–ª–æ–Ω—É, –Ω–∞–∑—ã–≤–∞–µ—Ç—Å—è —Ä–µ–≥—É–ª—è—Ä–Ω—ã–º –≤—ã—Ä–∞–∂–µ–Ω–∏–µ–º, –∞ —è–∑—ã–∫ Python –æ–±–µ—Å–ø–µ—á–∏–≤–∞–µ—Ç –ø–æ–¥–¥–µ—Ä–∂–∫—É —Ä–µ–≥—É–ª—è—Ä–Ω—ã—Ö –≤—ã—Ä–∞–∂–µ–Ω–∏–π —Å –ø–æ–º–æ—â—å—é –º–æ–¥—É–ª—è —Å—Ç–∞–Ω–¥–∞—Ä—Ç–Ω–æ–π –±–∏–±–ª–∏–æ—Ç–µ–∫–∏ `re`, –æ –∫–æ—Ç–æ—Ä–æ–º –º—ã —Ä–∞—Å—Å–∫–∞–∂–µ–º –≤ —Å–ª–µ–¥—É—é—â–µ–º —É—Ä–æ–∫–µ.

### –†–µ–∑—é–º–µ

–ó–Ω–∞–Ω–∏–µ —Ç–æ–≥–æ, –∫–∞–∫ –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è—Ç—å —Å—Ç—Ä–æ–∫–∏ –∏ –º–∞–Ω–∏–ø—É–ª–∏—Ä–æ–≤–∞—Ç—å –∏–º–∏, –æ—á–µ–Ω—å –≤–∞–∂–Ω–æ –¥–ª—è –ø—Ä–æ–≥—Ä–∞–º–º–∏—Å—Ç–æ–≤, –ø–æ—Å–∫–æ–ª—å–∫—É –Ω–∞–º —á–∞—Å—Ç–æ –ø—Ä–∏—Ö–æ–¥–∏—Ç—Å—è —Ä–∞–±–æ—Ç–∞—Ç—å —Å —Ç–µ–∫—Å—Ç–æ–≤–æ–π –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏–µ–π. –ú–∞–Ω–∏–ø—É–ª–∏—Ä–æ–≤–∞—Ç—å —Å—Ç—Ä–æ–∫–∞–º–∏ –≤ Python –º–æ–∂–Ω–æ —Å –ø–æ–º–æ—â—å—é —Ç–∞–∫–∏—Ö –æ–ø–µ—Ä–∞—Ü–∏–π –∫–∞–∫ —Å–ø–ª–∞–π—Å–∏–Ω–≥, –∏–Ω–¥–µ–∫—Å–∏—Ä–æ–≤–∞–Ω–∏–µ –∏ –Ω–∞—Ä–µ–∑–∫–∞, –∞ –º–æ–∂–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –±–æ–ª–µ–µ —â–µ–¥—Ä—ã–µ –º–µ—Ç–æ–¥—ã, –ø—Ä–µ–¥–æ—Å—Ç–∞–≤–ª—è–µ–º—ã–µ —Ç–∏–ø–æ–º string.
